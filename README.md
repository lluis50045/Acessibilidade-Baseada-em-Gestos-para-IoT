# Acessibilidade-Baseada-em-Gestos-para-IoT
O **Tom** é uma proposta de startup de impacto social desenvolvida como projeto final para a disciplina de **Sistemas Distribuídos**. O foco principal é a inclusão: permitir que pessoas com deficiência (PcD) ou limitações na fala consigam se comunicar e controlar dispositivos inteligentes (como a Amazon Alexa) através de comandos gestuais.
